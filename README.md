## 📖 About the Project

**Belajar LinkedIn – Mastering LinkedIn for Career Growth** adalah **landing page sederhana** untuk mempromosikan kelas online tentang **strategi optimasi profil, personal branding, dan networking di LinkedIn**.  

Website ini menampilkan struktur **hero section, about section, key benefits, testimonials placeholder, hingga call-to-action akhir** agar pengguna terdorong untuk mendaftar.  
Didesain dengan **clean, minimalis, responsif**, serta memanfaatkan **animasi ringan** agar lebih interaktif.

---

## 💡 Project Details
- Menyediakan landing page promosi kelas yang **jelas** dan **informatif**.  
- Memberikan gambaran singkat tentang kelas melalui **About Section**.  
- Menonjolkan **3–4 manfaat utama** dari kelas.  
- Menyediakan **placeholder testimonial alumni**.  
- Menampilkan **CTA button** untuk pendaftaran di beberapa bagian.  
- Responsif untuk tampilan **mobile hingga desktop**.  
- Menggunakan **efek interaktivitas sederhana** (hover & scroll animation).

## ✨ Landing Page Structure
- **Hero Section** → Judul kelas, subjudul singkat, CTA button **“Daftar Sekarang”**.  
- **About Section** → Deskripsi singkat kelas (100–150 kata).  
- **Key Benefits** → 3–4 manfaat utama mengikuti kelas.  
- **Testimonials** → Placeholder untuk feedback alumni.  
- **Final Call-to-Action** → Button pendaftaran di bagian akhir.

## 🚀 Getting Started

### 1) Clone repository
```bash
git clone https://github.com/zaki-ramadhan/belajar_linked_in.git
cd belajar_linked_in
```

### 2) Install dependencies
```bash
npm install
```

### 3) Run development server
```bash
npm run dev
```
Lalu buka di browser:  
👉 [http://localhost:5173](http://localhost:5173)
